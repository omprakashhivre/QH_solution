# Quick home solutions

Project - Quick home solutions ( Duration 02 Month(s) )


## Description

Problem for finding help for domestic services is always tedious when you
do not have any contacts. it is usually hard to find domestic trusted
workers. We can solve this problem by creating a website for a user to
connecting appropriate local services like electric work, home services
repairing, delivery , painting, colouring etc. other related works, probably
in low cost.
